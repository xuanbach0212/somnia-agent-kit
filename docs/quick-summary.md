## 🚀 Deploy in 3 Steps

### Step 1: GitBook Account (2 phút)
```
1. Go to https://www.gitbook.com
2. Sign up (free)
3. Confirm email
```

### Step 2: Import Repository (3 phút)
```
1. Click "New Space"
2. Choose "Import from GitHub"
3. Select: somnia-ai repo
4. Select: dev branch
5. Click "Import"
```

### Step 3: Done! 🎉
```
GitBook auto-syncs with GitHub
URL: your-space.gitbook.io
Share with team!
```
